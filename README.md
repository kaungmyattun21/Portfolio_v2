# Engineering Portfolio | Kaung Myat Tun

A high-performance, minimalist engineering portfolio built with Next.js 16 and React 19. Designed with a focus on clean architecture, performance, and rich user experience.

![Portfolio Preview](/og-image.png)

## 🚀 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Typography**: [Outfit (Google Fonts)](https://fonts.google.com/specimen/Outfit)
- **Deployment**: [Vercel](https://vercel.com/)

## ✨ Key Features

- **Rich Aesthetics**: Premium dark mode UI with glassmorphism effects and custom cursor interactions.
- **Dynamic Projects Section**: Smooth carousel interactions for project screenshots with hover-triggered transitions.
- **SEO Optimized**: Advanced metadata configuration, OpenGraph integration, and automated sitemap generation.
- **Responsive Design**: Mobile-first architecture ensuring seamless experiences across all device sizes.
- **Clean Code Architecture**: Modular component structure and decoupled data management (`projects.json`, `experience.json`).

## ⚡ Quick Commands

- `npm run dev` — Start development server
- `npm run build` — Build for production

## 📦 Deployment

The project is optimized for Vercel deployment. Simply connect your GitHub repository to Vercel, and it will auto-detect the Next.js settings.
